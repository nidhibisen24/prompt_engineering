Components of a Prompt

To get consistent results from an AI, you must stop viewing prompts as simple questions and start viewing them as **structured commands**. A professional prompt is composed of four distinct elements.

1 Instruction

The core task you want the model to perform. Be as specific as possible (e.g., "Analyze the sentiment" instead of "Tell me about this").

2 Context

External information or a persona the model should adopt. (e.g., "Respond as a medical researcher with 20 years of experience").

3 Input Data

The specific content the AI needs to process, often wrapped in tags like <text> for clarity.

4 Output Indicator

Formatting rules for the response. (e.g., "Provide the output in a JSON format" or "Keep the summary under 50 words").

Example

[Context] Acting as a Designer

[Instruction] extract the top 3 skills from the following resume.

[Input] Resume: "Expertise in UI, UX, and Designing..."

[Output] Format the results as a bulleted list.
